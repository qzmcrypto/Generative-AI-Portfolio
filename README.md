# 🎨 Generative AI Portfolio

A consolidated repository showcasing my Generative AI projects and experiments completed during my AI/ML training. This portfolio highlights my hands-on experience with various generative models, from text generation to advanced image stylization.

---

### 📂 Repository Structure

*   **Task 1: Text Generation (Richard III)** 
    *   **Description:** Training a text generation model using Shakespeare's *Richard III* dataset. 
    *   **Files:** `Task_1_Model_Trained.ipynb`, `richardIII.txt`

*   **Task 2: Image Generation**
    *   **Description:** Exploring generative models to synthesize and generate images.
    *   **Files:** `Task_2_Image_Generation.ipynb`

*   **Task 3: Poetry/Sonnet Generation**
    *   **Description:** Advanced text generation focused on poetic structures using a sonnet dataset.
    *   **Files:** `Task_3_Text_Generation.ipynb`, `sonnet.txt`

*   **Task 4: Pix2Pix Image Translation**
    *   **Description:** Implementing the Pix2Pix conditional GAN architecture for image-to-image translation.
    *   **Files:** `Task4_pix2pix_ImageTranslation.ipynb`

*   **Task 5: Neural Style Transfer**
    *   **Description:** Applying the artistic style of one image to the content of another using Neural Style Transfer (NST).
    *   **Files:** `Task5_NeuralStyleTransfer.ipynb`, `content.jpg`, `style.jpg`, `stylized_output.jpg`
